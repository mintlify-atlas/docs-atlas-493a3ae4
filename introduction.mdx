# Welcome to Chokmah

**Chokmah** (חָכְמָה) is a personal knowledge repository designed to serve as a second brain for web development learning resources. The name comes from the Hebrew word for "wisdom," representing the second sephirah in the Tree of Life of Kabbalah.

This documentation site curates essential concepts, practical examples, and reference materials across frontend, backend, and development tools that are important to have quick access to.
